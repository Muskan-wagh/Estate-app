# 🏡 Estate App

A modern real estate web application built with Next.js and Supabase that allows users to explore property listings and view detailed information about plots and flats. It also includes an admin system to post properties with images, videos, and blogs.

## Features

* Browse property listings
* View detailed property pages
* Image and video previews
* Admin panel to add properties
* Blog section for updates

## Tech Stack

Next.js • Supabase • Tailwind CSS

## Setup

git clone https://github.com/Muskan-wagh/estate-app.git
cd estate-app
npm install
npm run dev
